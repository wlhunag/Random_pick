Random_pick
===========

可自訂名單的 python 亂數程式，可跨平台使用

簡介：
---
-----------------------------------

班上有時候需要隨機亂數抽人，以前都是用 Excel 直接亂數排序

現在寫成小程式


安裝：
---
-----------------------------------

除了安裝 Python 以外，還要安裝PyQt4 


使用：
---
-----------------------------------

主程式為 Random_choice_portable_Mac.pyw


匯入名單的格式為學號 + <tab> + 姓名

直接從 excel 檔案貼上就是這種格式

抽人的時候，抽過的就不會再次出現，除非換名單或是重新啟動程式

新增或刪除名單以後，需要重新啟動程式，才會出現正確的名單
